# 🎯 小工具 - 独立待办清单 (Todo GUI)

一个**纯 Python + Tkinter** 开发的轻量级桌面待办清单工具，无需浏览器，独立小窗口运行，专为追求极致简洁和可爱体验的用户设计。

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ 功能特性

- **实时大时钟**：桌面右上角显示当前时间
- **可选截止日期**：每条待办可设置截止日，到期会高亮提醒
- **一键完成**：任务完成后自动从待办列表消失，并记入「功德簿」
- **功德簿弹窗**：随时查看所有已完成记录（带完成时间）
- **底部小花飘落动画**：完成任务时有可爱的小花飘落特效
- **独立小窗口**：无需安装浏览器插件，打开即用
- **数据自动保存**：关闭程序后下次打开自动恢复待办和历史记录
